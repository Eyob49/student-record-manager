# Student Record Manager

A simple **Python CLI application** to manage student records.  
This project demonstrates basic **OOP, file persistence, and CRUD operations**—a foundation for backend development.

## Features

- Add a new student
- View all students
- Update existing student information
- Delete a student
- Save and load data from a JSON file (`data.json`)

## Technology

- Python 3
- JSON for data storage
